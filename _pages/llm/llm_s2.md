# 使用LLM API开发应用

## 基本概念
temperature: 0-1
越接近0，则预测的随机性会越低，产生更保守、可以测的文本，不太可能生成意想不到或不寻常的词。
越接近1，则预测的随机性会越高，会产生更有创意、多样化的文本。
